# Enrollment Analytics Dashboard Design

## Purpose
Provide leadership with visibility into enrollment trends,
program demand, and funding impact.

---

## Dashboard Pages

### Page 1: Enrollment Overview
**Visuals**
- Line chart: Enrollment by Academic Term
- KPI Card: Total Active Students
- Bar chart: Enrollment by Program

**Business Questions**
- Is enrollment increasing or declining?
- Which programs have the highest demand?

---

### Page 2: Funding Analysis
**Visuals**
- Table: Student Funding Summary
- Bar chart: Funding by Program
- KPI Card: Total Funding Allocated

**Business Questions**
- How is funding distributed?
- Which programs require additional funding?

---

## Filters
- Academic Term
- Program
- Department

---

## Data Source
- FACT_ENROLLMENT table (Snowflake / SQL)
