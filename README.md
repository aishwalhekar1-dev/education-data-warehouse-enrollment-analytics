# education-data-warehouse-enrollment-analytics
Education# Education Data Warehouse & Enrollment Analytics

## Overview
This project demonstrates the design of an education-focused data warehouse to analyze student enrollment trends, funding impact, and data quality. The solution reflects real-world public sector and education systems, aligned with data governance and compliance requirements.

## Business Objective
Provide education leadership and finance stakeholders with accurate enrollment and funding insights to support strategic planning, budget allocation, and compliance reporting.

## Stakeholders
- Education Department Leadership
- Finance & Budget Teams
- Institutional Research
- Compliance & Audit Teams

## Key Business Questions
- How has student enrollment changed across academic terms?
- What is the impact of enrollment changes on funding allocation?
- Are there data quality issues that could affect reporting accuracy?
- How can standardized data validation improve decision-making?

## Data Sources
- Student master data
- Enrollment records by academic term
- Course and credit information
- Funding allocation data

> Note: This project uses sample and publicly available data for demonstration purposes.

## Data Warehouse Design
- Fact Table: Enrollment Fact
- Dimensions: Student, Course, Academic Term
- Source-to-target data mappings documented
- Data quality and validation rules applied

## Tools & Technologies
- SQL (Snowflake-style)
- Power BI / Tableau
- Excel
- Data Mapping & Data Validation
- GitHub for documentation

## Key KPIs
- Total Enrollment by Term
- Enrollment Growth Rate
- Funding per Student
- Credit Load Distribution
- Data Quality Exception Count

## Deliverables
- Source-to-target mapping documentation
- Data quality rules and validation checks
- SQL scripts for data transformation
- Enrollment analytics dashboard
- Business insights and recommendations

## Key Insights
- Enrollment trends directly impact funding allocation and resource planning.
- Data inconsistencies can result in inaccurate funding calculations.
- Standardized validation rules improve reporting accuracy and audit readiness.

## Future Enhancements
- Integration with real-time enrollment systems
- Automation of data quality checks
- Expansion to multi-year funding analysis
